# `@reforged/tsconfig`

A TypeScript Compiler configuration, used as part of ReForged project.

## Usage

Install this package as `devDependency` and set `extends` to `@reforged/tsconfig`
in your `tsconfig.json`, like so:

```jsonc
/* tsconfig.json */
{
    "extends": "@reforged/tsconfig",
    // (...)
}
```
